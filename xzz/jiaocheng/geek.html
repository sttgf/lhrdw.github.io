📢 使用前必读

  1. 仅支持 Windows XP 以上系统（不包含XP）
  2. 必须关闭杀毒软件或添加白名单
  3. 右键选择「以管理员身份运行」
    Win11需点击「更多操作」
  4. 遇到 SmartScreen 拦截请选择「仍要运行」

📦 版本说明

7z便携版

必须直接在压缩包内运行（不解压使用）

ZIP版

暂不提供下载

🛠️ 两种卸载模式

普通卸载
  1. 双击目标软件 或 右键选择「卸载(U)」
  2. 等待完成自带卸载流程
  3. 自动扫描残留文件/注册表（耗时较长）
  4. 勾选所有检测结果后清理
强制删除
  1. 右键选择「强制删除(F)」
  2. 直接删除主文件/注册表项
  3. 不会二次扫描残留文件
  4. 适合快速清理常规软件

❗ 注意事项

  • 卸载过程中所有安全警告请一律放行
  • 系统关键组件请勿强制删除
